# Security policy

## Supported versions

Cheatcode V2 is the only supported code line. Security fixes target the current
`main` branch and the production release derived from it; removed V1 and legacy
deployments are not maintained.

## Reporting a vulnerability

Report vulnerabilities through this repository's private vulnerability reporting
form under the **Security** tab. Do not open a public issue with exploit details,
credentials, user data, or provider keys.

Include the affected surface, reproducible steps, expected impact, and any safe
proof-of-concept material. Reports are acknowledged and triaged privately before a
coordinated fix and disclosure.
