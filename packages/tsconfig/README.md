# @cheatcode/tsconfig

Shared TypeScript configuration package for apps, Workers, and libraries.

## Public Exports

- `base.json`
- `library.json`
- `nextjs.json`
- `worker.json`

## Code Checks

This package has no runtime code. Config consumers validate it through their own
`tsc -p tsconfig.json --noEmit` checks.

## Env

None.
