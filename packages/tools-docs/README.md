# @cheatcode/tools-docs

Document-generation tools for the Cheatcode agent.

The package builds sandbox-side JavaScript for PPTX, DOCX, XLSX, and PDF generation. Generated bytes are uploaded through the request-scoped artifact runtime, which stores bytes in R2 and returns short-lived download URLs. Document-library dependencies belong to the sandbox's `/opt/cheatcode-doc-runtime`; this package emits references to them as source text and does not load them in the Worker bundle.

Sandbox execution and artifact persistence are accepted through the neutral
ports in `@cheatcode/sandbox-contracts`; this package does not depend on the code
tool implementation.

## Public Exports

- `executeGenerateSlides`
- `executeGenerateDocx`
- `executeGenerateXlsx`
- `executeGeneratePdf`
- Zod input/output schemas for each tool

## Code Checks

```bash
pnpm --filter @cheatcode/tools-docs typecheck
pnpm --filter @cheatcode/tools-docs lint
```

## Env

This package reads no environment variables directly. Agent-worker supplies the sandbox and artifact runtime.
